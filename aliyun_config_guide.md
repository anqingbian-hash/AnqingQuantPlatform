## 三、关键配置点

### 3.1 镜像选择（非常重要！）

**必须选择：**
- Ubuntu 22.04 LTS（推荐）
- 或 Ubuntu 20.04 LTS

**不要选择：**
- CentOS（已停止维护）
- Windows（成本高，不适合Python开发）
- 其他Linux发行版（兼容性未知）

### 3.2 存储选择

**必须选择：**
- SSD（固态硬盘）
- 容量：40G起步（可以后续扩容）

**不要选择：**
- HDD（机械硬盘，速度慢）

### 3.3 带宽选择

**建议：**
- 初期：3M（足够）
- 访问量增大后：升级到5M或10M

**说明：**
- 带宽可以后续升级
- 不需要一开始就很大

### 3.4 付费方式

**建议：**
- 包年包月（年付更便宜，通常节省20-30%）

**新用户优惠：**
- 阿里云通常有新用户优惠（首年3-5折）
- 建议抓住优惠机会

## 四、购买步骤

### 步骤1：选择地域
- 建议：华东1（杭州）或华东2（上海）
- 原因：国内访问速度快

### 步骤2：选择实例规格
- 建议：2核4G（方案A）或4核8G（方案B）
- 新用户通常有优惠，建议选择4核8G

### 步骤3：选择镜像（关键）
- 点击"公共镜像"
- 选择"Ubuntu"
- 选择"Ubuntu 22.04 LTS 64位"
- **这步非常重要！**

### 步骤4：选择存储
- 类型：选择"SSD云盘"
- 容量：40G
- **这步很重要！**

### 步骤5：选择带宽
- 选择"按使用量"或"按固定带宽"
- 建议选择"按固定带宽"
- 带宽：3M（方案A）或5M（方案B）

### 步骤6：设置密码（非常重要！）
- 登录凭证：选择"自定义密码"
- 用户名：root（默认）
- 密码：设置一个强密码（大小写字母+数字+符号，8位以上）
- **记住这个密码！**

### 步骤7：确认订单
- 检查所有配置
- 检查价格
- 点击"立即购买"
- 支付
