# A股数据源调研

## 已知数据源列表

### 1. 官方数据源
- **上交所官网** - Level-1行情（免费）
- **深交所官网** - Level-1行情（免费）
- **中国结算官网** - 股东数据

### 2. 第三方数据源

#### 免费数据源
1. **新浪财经API**
   - 实时行情：`http://hq.sinajs.cn/list=sh000001`
   - 优点：免费、稳定、快速
   - 缺点：数据有限

2. **腾讯财经API**
   - 实时行情：`http://qt.gtimg.cn/q=sh000001`
   - 优点：免费、稳定
   - 缺点：数据有限

3. **网易财经**
   - 历史数据、实时行情
   - 优点：数据全面
   - 缺点：需要爬虫

4. **东方财富网**
   - 实时行情、财务数据
   - 优点：数据丰富
   - 缺点：需要爬虫

5. **AKShare**（当前使用）
   - 综合性数据源
   - 优点：Python库，数据全面
   - 缺点：网络连接不稳定

#### 付费数据源
1. **Tushare Pro**
   - 专业数据
   - 优点：数据准确、稳定
   - 缺点：需要积分

2. **聚宽（JoinQuant）**
   - 量化数据
   - 优点：数据质量高
   - 缺点：付费

3. **米筐（RiceQuant）**
   - 量化数据
   - 优点：专业
   - 缺点：付费

4. **优矿（UQer）**
   - 量化数据
   - 优点：全面
   - 缺点：付费

### 3. 券商API
- **华泰证券** - OpenAPI
- **中信证券** - 极速交易
- **国泰君安** - 君弘APP
- 等多家券商提供API
