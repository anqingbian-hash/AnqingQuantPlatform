## 完整 API 列表

### 基础 API（3个）

1. `GET /` - 首页
2. `GET /health` - 健康检查
3. `GET /api/test` - API 测试

### 认证 API（5个）

1. `POST /api/auth/register` - 用户注册
2. `POST /api/auth/login` - 用户登录
3. `GET /api/auth/me` - 获取当前用户
4. `GET /api/users` - 用户列表
5. `GET /api/users/<id>` - 获取单个用户

### 核心 API（5个）

1. `GET /api/stocks` - 股票列表
2. `GET /api/stocks/<code>` - 获取股票
3. `POST /api/analyze` - 股票分析
4. `POST /api/scan` - 市场扫描
5. `GET /api/charts` - 图表数据（待实现）

### 导出 API（2个）

1. `POST /api/export/pdf` - 导出 PDF
2. `POST /api/export/feishu` - 导出飞书文档

**总计**: 15 个 API

## 数据模型

### User 模型（用户）
- id: 主键
- username: 用户名（唯一）
- email: 邮箱（唯一）
- password_hash: 密码哈希
- role: 角色（admin/user/guest）
- is_active: 是否激活
- created_at: 创建时间
- updated_at: 更新时间
- last_login_at: 最后登录时间
- login_count: 登录次数

### Stock 模型（股票）
- id: 主键
- code: 股票代码（唯一）
- name: 股票名称
- market: 市场（SH/SZ）
- industry: 行业
- sector: 板块
- created_at: 创建时间

## 技术架构

### 完整架构

```
run_server.py（服务器启动）
    ↓
app/__init__.py（应用工厂 v9）
    ↓
AuthService（认证服务）
    ↓
ExportService（导出服务）
    ↓
SQLAlchemy（数据库 ORM）
    ↓
SQLite（数据库）
    ↓
Flask 应用
    ↓
15个 API 接口
```

### 技术栈

- **后端框架**: Flask 3.0.0
- **数据库**: SQLite
- **ORM**: SQLAlchemy
- **认证**: JWT (PyJWT) + Werkzeug 密码哈希
- **配置**: YAML
- **导出**: 文本文件 + Markdown

### 安全特性

1. **密码安全**: Werkzeug 密码哈希
2. **Token 安全**: JWT token 认证
3. **权限控制**: 装饰器权限验证
4. **数据隔离**: 双环境完全隔离
