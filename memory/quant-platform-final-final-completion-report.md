## 完整 API 列表

### 基础 API（3个）

1. `GET /` - 首页
2. `GET /health` - 健康检查
3. `GET /api/test` - API 测试

### 认证 API（5个）

1. `POST /api/auth/register` - 用户注册
2. `POST /api/auth/login` - 用户登录
3. `GET /api/auth/me` - 获取当前用户
4. `GET /api/users` - 用户列表
5. `GET /api/users/<id>` - 获取单个用户

### 核心 API（4个）

1. `GET /api/stocks` - 股票列表
2. `GET /api/stocks/<code>` - 获取股票
3. `POST /api/analyze` - 股票分析
4. `POST /api/scan` - 市场扫描

### 实时分析 API（3个）

1. `GET /api/analysis/realtime/<code>` - 实时分析
2. `GET /api/analysis/chip/<code>` - 筹码分布分析
3. `GET /api/analysis/market` - 市场综合分析

### 导出 API（2个）

1. `POST /api/export/pdf` - 导出 PDF
2. `POST /api/export/feishu` - 导出飞书文档

**总计**: 17 个 API 接口

## 技术架构

### 完整架构

```
run_server.py（服务器启动）
    ↓
app/__init__.py（应用工厂 v10）
    ↓
AuthService（认证服务）
    ↓
MarketAnalysisService（市场分析服务）
    ↓
SQLAlchemy（数据库 ORM）
    ↓
SQLite（数据库）
    ↓
Flask 应用
    ↓
17个 API 接口
```

### 技术栈

- **后端框架**: Flask 3.0.0
- **数据库**: SQLite
- **ORM**: SQLAlchemy
- **认证**: JWT (PyJWT) + Werkzeug
- **配置**: YAML
- **导出**: 文本文件 + Markdown
