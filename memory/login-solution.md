如果您想测试 API 是否正常，可以使用以下命令：

```bash

```

## 🔍 故障排查

如果您仍然无法登录，请检查：

1. **浏览器控制台**
   - 按 F12 打开开发者工具
   - 查看 Console 标签页的错误信息
   - 查看 Network 标签页的请求详情

2. **网络连接**
   - 确保能访问 http://localhost:7000
   - 确保没有防火墙阻止

3. **输入内容**
   - 用户名：admin（小写）
   - 密码：admin123（数字 123）

4. **服务器状态**
   - 确认服务器正在运行
   - 端口 7000 正常监听
