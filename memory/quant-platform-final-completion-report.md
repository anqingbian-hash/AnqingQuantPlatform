## 数据模型

### User 模型（用户）

- id: 主键
- username: 用户名
- email: 邮箱
- password_hash: 密码哈希
- role: 角色
- is_active: 是否激活
- created_at: 创建时间
- last_login_at: 最后登录时间
- login_count: 登录次数

### Stock 模型（股票）

- id: 主键
- code: 股票代码
- name: 股票名称
- market: 市场（SH/SZ）
- industry: 行业
- sector: 板块

## 技术架构

### 当前架构

```
run_server.py（服务器启动）
    ↓
app/__init__.py（应用工厂 v7）
    ↓
AuthService（认证服务）
    ↓
SQLAlchemy（数据库 ORM）
    ↓
SQLite（数据库）
    ↓
Flask 应用
    ↓
基础 API + 认证 API + 核心 API
```

### 数据库

- **类型**: SQLite
- **位置**: `data/test/quant_test.db`
- **表**: users, stocks

### 认证系统

- **密码哈希**: Werkzeug
- **JWT Token**: PyJWT
- **权限控制**: 装饰器
