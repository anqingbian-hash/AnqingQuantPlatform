## 📝 问题记录

### 问题1：技能注册表未检测到技能

**问题**：技能注册表返回"总计: 0 个技能"

**原因**：技能文件嵌套在子目录中（例如：`01-长期记忆---{uuid}/01-长期记忆/`）

**影响**：技能无法被自动注册

**解决方案**：
1. 优化技能注册表，支持递归扫描
2. 跟踪到子目录中的技能
3. 自动跳过中间目录层级

### 问题2：技能加载失败

**问题**：技能适配器无法正确执行技能

**原因**：技能路径结构复杂，需要特殊处理

**影响**：技能执行失败

**解决方案**：
1. 增强技能适配器的路径处理
2. 支持多种目录结构
3. 提供更好的错误信息
