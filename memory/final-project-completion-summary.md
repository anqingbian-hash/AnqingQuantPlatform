## 技术架构

### 技术栈

- **后端框架**: Flask 3.0.0
- **数据库**: SQLite（测试）/ MySQL（生产）
- **ORM**: SQLAlchemy
- **认证**: JWT (PyJWT) + Werkzeug
- **配置**: YAML
- **导出**: 文本文件 + Markdown

### 安全特性

1. **密码安全**: Werkzeug 密码哈希
2. **Token 安全**: JWT token 认证
3. **权限控制**: 装饰器权限验证
4. **数据隔离**: 双环境完全隔离

### 部署方式

- **测试环境**: Flask 开发服务器
- **生产环境**: Gunicorn + Nginx
- **进程管理**: Supervisor
- **反向代理**: Nginx
