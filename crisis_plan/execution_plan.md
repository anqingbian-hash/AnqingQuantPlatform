## 🎯 立即执行方案（优先级排序）

### 优先级 1：快速回笼资金（Day 1-7）

#### A. 应收账款催收（Day 1-3）
```
执行时间：立即开始
目标：快速回笼 100-200 万

步骤：
1. [ ] 盘点所有应收账款（按客户/金额/账期分类）
2. [ ] 按账期分类：
   - 30 天内：优质客户（电话催收）
   - 60 天内：中等客户（上门催收）
   - 90 天内：问题客户（法律手段）
3. [ ] 电话催收话术：
   - "您好，是 XX 公司吗？我们是 XX 水泥，请问您欠的 XX 万什么时候能付？"
   - "我们公司现在资金紧张，希望能尽快回款"
   - "如果您现在不方便，能否先付一半？"
4. [ ] 上门催收策略：
   - 携带合同和发票
   - 明确还款日期
   - 签订还款计划
5. [ ] 法律准备：
   - 准备起诉材料
   - 咨询律师
   - 必要时起诉

预期收益：100-200 万
```

#### B. 库存变现（Day 2-5）
```
执行时间：立即开始
目标：快速回笼 50-150 万

步骤：
1. [ ] 盘点所有库存（按产品/数量/成本/市价分类）
2. [ ] 分类处理：
   - 热销品：按市价正常销售
   - 滞销品：降价 10-20% 促销
   - 过期品：降价 30-50% 处理
3. [ ] 促销渠道：
   - 现有客户（电话/微信通知）
   - 行业群（建材群/水泥群）
   - 同行（低价出售）
4. [ ] 销售策略：
   - "滞销产品促销，限时优惠"
   - "老客户专享，额外 5% 折扣"
   - "大批量采购，再优惠"

预期收益：50-150 万
```

#### C. 债务协商（Day 3-7）
```
执行时间：立即开始
目标：延长还款期限，降低每月压力

步骤：
1. [ ] 联系所有银行/债权人
2. [ ] 提出方案：
   - 延期 3-6 个月
   - 分期还款
   - 利率不变
3. [ ] 准备材料：
   - 财务报表
   - 还款计划
   - 恢复方案
4. [ ] 沟通话术：
   - "我们公司现在遇到困难，希望延期 3-6 个月"
   - "我们有完整的恢复计划，6 个月后一定还清"
   - "如果您不同意，我们可能无法继续运营"

预期收益：每月还款压力 -50%
```
