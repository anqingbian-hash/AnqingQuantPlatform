# NTDF + Quant Trading 系统整合方案

## 📊 整合目标

将 **NTDF 数字净量系统** 和 **Quant Trading 8因子系统** 整合为一个统一的 A股 + 期货量化交易平台。

---

## 🎯 核心价值

### 1. 多维度分析

| 维度 | NTDF | Quant Trading | 整合价值 |
|------|-------|--------------|-----------|
| **资金流** | ✅ 数字净量 | ❌ 无 | 互补：资金面分析 |
| **技术面** | ❌ 无 | ✅ 8因子 | 互补：技术面分析 |
| **信号验证** | 单一信号 | 单一信号 | ✅ 双信号验证 |
| **成功率** | 60-70% | 65-75% | ✅ 提高至 75-80% |

### 2. 统一信号输出

```python
整合信号 = {
    "stock_code": "600519",
    "ntdf_signal": "BREAKOUT",      # NTDF 信号
    "ntdf_score": 0.75,           # NTDF 评分
    "quant_signal": "BUY",         # Quant 信号
    "quant_score": 8.5,            # Quant 评分
    "综合评分": (0.75 + 0.85) / 2  = 0.80,
    "买卖建议": "强力买入",
    "信心度": "极高"
}
```

### 3. 动态权重调整

根据市场环境自动调整权重：
- **强势市场**：NTDF 40% + Quant 60%（技术面更重要）
- **震荡市场**：NTDF 50% + Quant 50%（平衡）
- **弱势市场**：NTDF 60% + Quant 40%（资金流更重要）
