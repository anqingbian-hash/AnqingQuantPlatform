### 5.1 形态检测算法

#### BC（购买高潮）检测算法
```python
# BC形态特征
# 1. 价格突破近期高点（上部区域高点密集）
# 2. 净量异常增大（做多/做空比 > 2）
# 3. DELTA斜率向上并加速（动能增强）

confidence_score = 0
if high_edge_count > 1000:
    confidence_score += 30
if net_long_ratio > 0.67 and surge_ratio > 2:
    confidence_score += 40
if delta_slope == 'up' and accelerating:
    confidence_score += 30

detected = confidence_score >= 70
```

#### SC（卖出密集区）检测算法
```python
# SC形态特征
# 1. 价格跌破近期低点（下部区域低点密集）
# 2. 净量爆发（做空/做多比 > 2）
# 3. DELTA斜率向下并加速（动能增强）

confidence_score = 0
if low_edge_count > 1000:
    confidence_score += 30
if net_short_ratio > 0.67 and spike_ratio > 2:
    confidence_score += 40
if delta_slope == 'down' and accelerating:
    confidence_score += 30

detected = confidence_score >= 70
```

#### LPS（最后的支撑点）检测算法
```python
# LPS形态特征
# 1. 价格站稳S2上方3根K线（企稳）
# 2. 净量净多稳定
# 3. DELTA平坦或向上

confidence_score = 0
if stabilized and above_support:
    confidence_score += 40
if net_long_stable:
    confidence_score += 35
if delta_slope in ['up', 'flat']:
    confidence_score += 25

detected = confidence_score >= 70
```

#### LPSY（最后的供应点）检测算法
```python
# LPSY形态特征
# 1. 价格在R2下方企稳3根K线（震荡）
# 2. 净量净空主导或由多转空
# 3. DELTA平坦或向下

confidence_score = 0
if consolidated and below_resistance:
    confidence_score += 40
if net_short_dominant or turning_from_long_to_short:
    confidence_score += 35
if delta_slope in ['down', 'flat']:
    confidence_score += 25

detected = confidence_score >= 70
```
