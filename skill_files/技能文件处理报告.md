## 🎯 技能功能概述

### 1. 长期记忆
- **功能**：AI 记忆系统，支持长期上下文存储
- **核心**：ai_memory.py
- **依赖**：Python 3.7+
- **特点**：无 API 密钥，本地存储

### 2. 联网搜索
- **功能**：安全的网络搜索工具，支持多个搜索引擎
- **核心**：web_search.py
- **依赖**：duckduckgo-search, beautifulsoup4, requests
- **特点**：DuckDuckGo + Wikipedia，24 小时缓存

### 3. 图像识别
- **功能**：安全的图片识别工具，支持本地和 API 两种模式
- **核心**：vision_ai.py
- **依赖**：openai/anthropic（API模式）或 torch/transformers（本地模式）
- **特点**：GPT-4o/Claude API 或本地模式（隐私保护）

### 4. 自动迭代
- **功能**：安全的自动任务迭代和优化系统
- **核心**：iteration_agent.py
- **依赖**：Python 3.7+（仅标准库）
- **特点**：质量评估 + 自动重试，支持工具白名单

### 5. 内容总结
- **功能**：内容总结和审阅系统
- **核心**：summary_review.py
- **依赖**：待确认
- **特点**：自动总结 + 审阅

### 6. AI 总结
- **功能**：基于 LLM 的内容总结
- **核心**：summary_review_llm.py
- **依赖**：待确认（可能是 openai/anthropic）
- **特点**：LLM 驱动的高质量总结

### 7. 互联网访问
- **功能**：为 AI Agent 提供完整的互联网访问能力
- **核心**：core.py + channels/
- **依赖**：多种第三方库
- **支持平台**：Twitter、YouTube、Reddit、小红书、B站等
- **特点**：零 API 费用，Cookie 加密存储，安全审计
