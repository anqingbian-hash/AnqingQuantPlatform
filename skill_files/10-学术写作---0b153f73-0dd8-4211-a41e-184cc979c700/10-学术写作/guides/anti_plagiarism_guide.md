# 降重技巧完整指南

## 🎯 降重核心原则

**核心公式**: 理解 + 重写 + 扩展 + 创新 = 原创内容

---

## 📋 降重技巧大全

### 技巧1: 同义词替换

**示例**:
```
原文: 人工智能技术在医疗领域有着广泛的应用。
降重: AI技术在医疗行业展现出巨大的运用潜力。
```

**强度**: ⭐ (轻度)
**优点**: 简单快速
**缺点**: 效果有限

---

### 技巧2: 句式变换

#### 2.1 被动转主动

```
原文: 实验结果被记录在表1中。
降重: 表1展示了实验结果。
```

#### 2.2 否定变肯定

```
原文: 这项技术不仅有效。
降重: 这项技术十分有效。
```

#### 2.3 疑问变陈述

```
原文: 是否可以提高效率？
降重: 这显然能够提高效率。
```

**强度**: ⭐⭐ (轻度)
**优点**: 保持原意
**缺点**: 语法变化不大

---

### 技巧3: 句子重组

#### 3.1 改变语序

```
原文: 因为采用了新方法，所以效率提高了。
降重: 效率的提升得益于新方法的采用。
```

#### 3.2 拆分长句

```
原文: 人工智能技术通过分析大量医学影像数据，能够辅助医生进行更准确的疾病诊断。
降重: 人工智能技术能够分析医学影像数据。通过这种方式，它辅助医生进行更准确的疾病诊断。
```

#### 3.3 合并短句

```
原文: 研究表明，这项技术有效。实验证实了这一点。
降重: 研究和实验都证实了这项技术的有效性。
```

**强度**: ⭐⭐⭐ (中度)
**优点**: 显著改变句式
**缺点**: 可能影响流畅性

### 技巧4: 扩展细节

**示例**:
```
原文: 深度学习算法可以用于疾病诊断。
降重: 深度学习算法通过分析患者的医学影像数据（如CT扫描、MRI图像等），能够识别出肉眼难以察觉的微小病变特征。以肺部结节检测为例，算法可以在5mm以下的结节识别中达到95%以上的准确率，远超人工阅片水平。
```

**强度**: ⭐⭐⭐ (中度)
**优点**: 增加内容深度
**缺点**: 增加篇幅

---

### 技巧5: 添加实例

**示例**:
```
原文: 深度学习在图像识别领域取得了显著成果。
降重: 深度学习在图像识别领域取得了显著成果。以ResNet为例，该网络在ImageNet挑战赛上将错误率降低到了3.57%，超越了人类水平。此外，在医疗影像诊断中，基于深度学习的系统对皮肤癌的识别准确率已达到专业医生水平，在皮肤病AI诊断大赛中获得第一名。
```

**强度**: ⭐⭐⭐⭐ (中高度)
**优点**: 提高可信度
**缺点**: 需要查找实例

---

### 技巧6: 改变逻辑结构

**示例**:
```
原文: 首先收集数据，然后清洗数据，最后训练模型。
降重: 模型训练的前置步骤是数据清洗，而数据清洗之前需要完成数据收集工作。这三个步骤构成了机器学习项目的数据准备阶段，缺一不可。
```

**强度**: ⭐⭐⭐⭐ (中高度)
**优点**: 彻底改变结构
**缺点**: 需要理解逻辑

---

### 技巧7: 改变视角

**示例**:
```
原文: 从技术角度来看，这个系统运行效率很高。
降重: 就用户体验而言，该系统的响应速度令人满意。
```

**强度**: ⭐⭐⭐⭐ (中高度)
**优点: 完全不同角度
**缺点**: 需要转换思维

---

### 技巧8: 类比和举例

**示例**:
```
原文: 机器学习算法可以自动从数据中学习规律。
降重: 机器学习算法类似于人类从经验中学习的过程。就像儿童通过多次接触逐渐认识物体一样，算法通过大量训练数据的迭代优化，自动识别出数据中的复杂模式和规律。
```

**强度**: ⭐⭐⭐⭐⭐ (高度)
**优点**: 生动形象
**缺点**: 增加篇幅

### 技巧9: 数据驱动表述

**强度**: ⭐⭐⭐ (中度)

### 技巧10: 综合降重（推荐）

**组合多个技巧**:
```python
from tools.plagiarism_reducer import PlagiarismReducer

reducer = PlagiarismReducer()

# 综合降重（高强度）
rewritten = reducer.reduce_all(
    original_text,
    intensity=0.8,  # 80%强度
    techniques=[
        "synonym_replace",      # 同义词替换
        "change_voice",         # 被动转主动
        "restructure_sentence", # 句式重组
        "expand_paragraph",     # 扩展细节
        "add_examples"          # 添加实例
    ]
)
```

**强度**: ⭐⭐⭐⭐⭐ (最高)
**优点**: 效果最好
**缺点**: 需要更多时间

### 目标标准

| 文档类型 | 查重率 | 技巧组合 |
|---------|-------|---------|
| 一般课程论文 | < 30% | 同义词+句式变换 |
| 本科毕业论文 | < 25% | +句子重组 |
| 硕士论文 | < 20% | +扩展+实例 |

| 期刊论文 | < 10% | 综合降重 |

## ⚠️ 注意事项

### ✅ 正确做法

1. **理解原文** - 深刻理解后再重写
2. **表达方式** - 用自己的话表达
3. **增加见解** - 加入自己的思考
4. **引用标注** - 标明来源

### ❌ 错误做法

1. ❌ 简单替换词语
2. ❌ 改变句子顺序
3. ❌ 翻译后翻译回来
4. ❌ 抄袭整段内容
