# NTDF数字净量分析教程学习笔记 - 2026-03-04

## 核心学习内容

### 一、系统核心理念
**基于威科夫时空价量交易逻辑的数据分析工具**
- **核心作用**：帮助普通交易者看资金方向、找买卖参考区间
- **定位**：不直接给买卖信号，只做数据分析辅助
- **特色**：全程贴合小白交易习惯，把复杂的专业逻辑转化为简单的看盘、用盘方法

### 二、系统核心功能解析

#### 1. 专业术语大白话转换（零基础入门关键）

| 系统术语 | 零基础大白话解读 | 核心作用 |
|---------|----------------|---------|
| **Wyckoff威科夫** | 一套成熟的"看资金、找点位"的交易思路系统 | 底层逻辑，不用深研，跟着系统指标看就行 |
| **时空价量（四大要素）** | 时间周期、涨跌空间、价格走势、资金量能 | 系统分析的四大核心，已帮咱们整合好 |
| **多周期组合** | 同时看小周期、大周期的盘面（如5分钟+15分钟） | 看大周期定方向，小周期找点位，减少判断失误 |
| **净量** | 实时"做多资金"和"做空资金"的差额 | 红色/正数=做多的人多，绿色/负数=做空的人多 |
| **Delta（DELTA BAR）** | 资金量的实时走势线 | 看资金和价格是否"匹配"，匹配则趋势稳，不匹配则可能变盘 |
| **SR线（动态滤网）** | 系统自画的"支撑线、压力线" | 红线=压力位（涨到这里大概率回落），绿线=支撑位（跌到这里大概率反弹） |
| **LPS（最后的支撑点）** | 下跌后，跌不动了的"关键支撑位" | 低吸的参考点位 |
| **LPSY（最后的供应点）** | 上涨后，涨不动了的"关键压力位" | 高抛的参考点位 |
| **BC（购买高潮）** | 买盘一下子冲到顶，后续大概率跌的点位 | 短期高点，别追多，可考虑做空 |
| **SC（卖出密集区）** | 卖盘一下子砸到底，后续大概率高涨的点位 | 短期低点，别追空，可考虑做多 |
| **TEST（测试）** | 价格碰到SR线/BC/SC，没跌破/没突破 | 测试成功=趋势继续，测试失败=趋势反转 |
| **顶背离/底背离** | 价格涨了但资金没涨（顶背离），价格跌了但资金没跌（底背离） | 顶背离=要跌，底背离=要涨 |

#### 2. 三大核心界面（一眼看懂）

**主图**：看价格+系统自画的"关键点位/线"
- **核心显示**：股票/期货的价格走势K线 + 自动画的SR线（支撑/压力） + 标注的LPS/LPSY/BC/SC
- **小白用法**：红色线=压力位，绿色线=支撑位，价格在支撑位附近就关注多，压力位附近就关注空

**副图1 - 数字净量（核心中的核心，看资金方向）**
- **核心显示**：数字数值 + 正负号 + 颜色（正数/红色=净多，负数/绿色=净空）
- **小白用法**：只看数字和颜色，不用算！红色正数越大，说明当下做多的资金越多，多方占优；绿色负数越大，说明当下做空的资金越多，空方占优

**副图2 - DELTA BAR（资金走势线，看资金和价格匹配度）**
- **核心显示**：一条实时波动的曲线（资金走势），和主图的价格走势对应
- **小白用法**：对比主图价格和这个资金曲线
  - 价格涨，资金曲线也涨→匹配，多头趋势稳，可持有多单
  - 价格跌，资金曲线也跌→匹配，空头趋势稳，可持有空单
  - 价格涨，资金曲线跌→不匹配（顶背离），要跌，别追多
  - 价格跌，资金曲线涨→不匹配（底背离），要涨，别追空

### 三、系统核心规则

#### 1. 多周期组合看盘（小白推荐）
- **新手必用周期组合**：15分钟（定大方向）+ 5分钟（找买卖点位）

**第一步：看15分钟周期（定方向）**
- 价格在SR绿线上方→大方向做多，只找多单点位
- 价格在SR红线下方→大方向做空，只找空单点位

**第二步：看5分钟周期（找精准点位）**
- 大方向做多：在5分钟主图找LPS支撑位/SC卖出密集区，价格碰到这些点位且5分钟净量是红色→可以关注多单
- 大方向做空：在5分钟主图找LPSY压力位/BC购买高潮，价格碰到这些点位且5分钟净量是绿色→可以关注空单

#### 2. 共振突破法则（只做准信号）
**什么是"共振突破"？（小白记死：满足3个条件，就是真信号）**
1. **价格突破**：价格突破/跌破主图的SR线（支撑/压力）
2. **净量匹配**：副图1的净量是红/绿（和突破方向一致，突破压力位净量红，跌破支撑位净量绿）
3. **资金同步**：副图2的DELTA资金曲线和价格同步突破/跌破（不背离）

**→ 3个条件同时满足=共振突破=真信号，可以做单；少一个条件=假信号，坚决不做！**

#### 3. 四不开仓原则（小白保命法则）
1. **不突破/不回踩，不开仓**：价格突破SR线后，要有小幅回踩，等回踩后再做单
2. **没设止损位，不开仓**：做单前先定好"亏多少就走"，止损位看系统主图的SR线
3. **止损位太大，不开仓**：亏损超过本金的1%-2%，就别做这个单
4. **涨跌空间太小，不开仓**：价格突破后，到下一个压力/支撑位的空间要≥止损位的3倍

**仓位控制**：单次做单仓位别超过本金的5%，轻仓是保命关键

### 四、系统核心策略

#### 1. 共振突破实操找法
**做多单（大方向多方）**：找3个共振
1. 15分钟周期：价格在绿线支撑上方，净量红色
2. 5分钟周期：价格向上突破主图红色压力位（或碰到LPS/SC）
3. 5分钟副图：净量红色正数，DELTA曲线和价格一起涨（不背离）

**做空单（大方向空方）**：找3个共振
1. 15分钟周期：价格在红线压力下方，净量绿色
2. 5分钟周期：价格向下跌破主图绿色支撑位（或碰到LPSY/BC）
3. 5分钟副图：净量绿色负数，DELTA曲线和价格一起跌（不背离）

#### 2. 止损止盈规则
- **止损**：直接用系统的SR线
  - 做多单：止损=最近的绿色支撑线下方1-2个点（跌破就平多单）
  - 做空单：止损=最近的红色压力线上方1-2个点（突破就平空单）

- **止盈**：
  - **新手首选**：看系统下一个SR线（最省心）
    - 做多单：止盈位设为下一个红色压力线下方（涨到压力位就平多，不贪）
    - 做空单：止盈位设为下一个绿色支撑线上方（跌到支撑位就平空，不贪）
  - **进阶方法**：赚3倍止损就走（如止损亏50点，赚到150点就平单）

### 五、小白实操流程（10分钟完整做单步骤）

1. **打开系统，选周期组合（15分钟+5分钟）**
2. **看15分钟周期，定大方向**
3. **看5分钟周期，找共振突破点位**
4. **设止损和止盈（按系统SR线）**
5. **开仓（轻仓，仓位≤5%）**
6. **做单后盯盘，浮盈保本**
7. **复盘（新手必做）**

### 六、小白避坑指南（5个核心禁忌）
1. **别只看一个指标做单**：必须看"价格+净量+DELTA"三者匹配
2. **别做反方向的单**：大方向是空方，就别做多
3. **别频繁做单**：系统的共振信号不多，一天能有1-2个就不错了
4. **别追涨杀跌**：价格涨了一大截再追多，跌了一大截再追空
5. **别不信系统的背离信号**：价格和DELTA曲线背离，哪怕感觉还要涨/跌，也别做单

### 七、学习建议
**零基础快速上手练习方法（3天练熟）**
1. **第一天**：认界面+记专业词，只看15分钟+5分钟，标出主图的SR线、副图的净量/DELTA
2. **第二天**：找共振突破点位，在模拟盘上找符合"3个共振"的点位
3. **第三天**：模拟盘做单，按完整流程做1-2单，严格遵守四不开仓和止损止盈
