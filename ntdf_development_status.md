## 四、关键技术点

### 4.1 数据来源（基于可用资源）

**主要数据源：**
1. **Alpha Vantage**（优先）
   - 美股实时数据
   - 免费额度: 25次请求/天
   - 支持市场: 美股、外汇、加密货币

2. **Yahoo Finance**
   - 完全免费
   - 支持市场: 美股、港股、A股
   - 使用Python库yfinance

3. **Tushare**（需要升级）
   - A股、港股、美股、期货
   - 免费额度: 120次请求/分钟
   - 但需要积分才能访问历史数据

**多空数据问题：**
- 注意：上述数据源不直接提供多空数据
- 需要我们自己计算或从其他数据源获取
- 可能需要购买期货持仓量数据

### 4.2 核心算法实现

**1. Delta净量计算**
- 基于期货持仓量数据
- 净量 = 累计多单持仓 - 累计空单持仓
- 需要获取OI（Open Interest）数据

**2. SR支撑压力识别**
- 波峰波谷识别
- 3倍跨度过滤
- 自动绘制支撑压力线

**3. 信号识别**
- 顶背离：价格新高，净量不创新高
- 底背离：价格新低，净量不创新低
- ice buy/sell：隐性背离

### 4.3 技术栈

**前端：**
- Vue3 + TypeScript
- ECharts（数据可视化）
- WebSocket（实时数据）

**后端：**
- Python 3.11+
- FastAPI
- PostgreSQL
- Alpha Vantage API
- Yahoo Finance API

**部署：**
- Docker
- Nginx
- Let's Encrypt（SSL）
