## 一、系统概述

### 1.1 项目背景
- **项目名称**：君居合智金WKF时空选股系统
- **核心目标**：基于威科夫（WKF）时空价量交易逻辑 + NTDF数字净量，实现智能选股分析
- **设计理念**：不直接给买卖信号，只做数据分析辅助，由交易员最终决策
- **上级领导**：卞董（董事长）

### 1.2 系统架构
```
WKF系统架构
├── 输入层（文华财经截图）
│   ├── 周线图
│   ├── 日线图
│   ├── 1小时图
│   ├── 15分钟图
│   └── 5分钟图
├── 分析层（4大Agent团队）
│   ├── Agent 1: 图像识别（OCR + 颜色分析）
│   ├── Agent 2: SR线检测（支撑/阻力线 + Y坐标映射）
│   ├── Agent 3: 净量/DELTA分析（多空动能 + 动能斜率）
│   └── Agent 4: WKF规则引擎（多周期共振 + 顶部/底部形态）
└── 输出层（决策建议）
    ├── 周期阶段（吸筹/拉升/派发/震荡）
    ├── 交易信号（做多/做空/观望/规避）
    ├── 仓位建议（0% - 20%）
    └── 风险等级（低/中/高）
```

## 二、核心功能模块

### 2.1 已完成功能

#### ✅ 图像识别模块
- **OCR引擎**：Tesseract 5.3.2（已安装）
- **颜色识别**：精准识别SR线、现价线的颜色阈值
  - R2（阻力线）：白色 BGR=[220-255, 220-255, 220-255]
  - S2（支撑线）：黄色 BGR=[20-60, 150-200, 150-200]
  - 现价线：黄色 BGR=[50-80, 210-215, 210-215]
- **版本迭代**：7个版本（V1-V7），准确率持续优化

#### ✅ SR线检测模块
- **检测算法**：Canny边缘检测 + 霍夫变换 + 颜色分类
- **Y坐标映射**：5个周期独立线性回归模型
  - 周线：价格 = 0.024471 × Y + 14.134588 (R²=0.6558)
  - 日线：价格 = 0.007228 × Y + 15.918152 (R²=0.2940)
  - 1小时：价格 = 0.006935 × Y + 16.273932 (R²=0.6884)
  - 15分钟：价格 = 0.001068 × Y + 17.529682 (R²=0.4034)
  - 5分钟：价格 = 0.000485 × Y + 17.880777 (R²=0.2500)
- **识别成果**：
  - S2（支撑线）：5个周期全部识别成功 ✅
  - 现价（黄色线）：5个周期全部识别成功 ✅
  - R2（阻力线）：需要优化（K线链接线干扰）

#### ✅ 净量/DELTA分析模块
- **净量分析**：
  - 红色像素 = 做多资金
  - 绿色像素 = 做空资金
  - 判断状态：净多/净空/中性
- **DELTA分析**：
  - 动能曲线检测（霍夫线变换）
  - 动能斜率判断：向上/向下/平坦
  - 顶背离/底背离识别

#### ✅ WKF规则引擎
- **第一铁律**（严格执行）：
  - 周线派发 = 全局无效（0分，0%仓位）
  - 日线派发 = 小周期无效（最高20分，0%仓位）
- **多周期共振**：
  - 五周期：周线、日线、1小时、15分钟、5分钟
  - 各20分：吸筹期/拉升期 = 20分；派发期 = 0分
  - 共振阈值：≥80分 = 高确定性有效；<80分 = 无效
- **周期阶段判定**：
  - **吸筹期**：跌幅≥50%/30%，站稳S2≥3根K线，箱体振幅≤8%，净多增长，动能向上
  - **拉升期**：突破箱体≥3%，回踩不跌破S2，高点抬升，净多稳定，动能正值且向上
  - **派发期**：跌破S2、高点拐头、振幅扩大、净空主导、动能向下、顶背离

#### ✅ 顶部/底部形态检测
- **顶部形态**：
  - **BC（购买高潮）**：短期高点，别追多
  - **LPSY（最后的供应点）**：二卖位置，高抛参考
- **底部形态**：
  - **SC（卖出密集区）**：短期低点，别追空
  - **LPS（最后的支撑点）**：二买位置，低吸参考

### 2.2 正在优化功能

#### ⏳ R2（阻力线）识别优化
- **当前问题**：
  - K线链接线干扰（白色像素占比低：0.50）
  - 霍夫变换阈值过高（threshold=100）
- **优化方案**：
  1. 放宽白色阈值到[200-255]
  2. 降低霍夫变换参数（threshold=20, minLineLength=width*0.1）
  3. 排除K线链接线（长度>0.3*width）

#### ⏳ 形态识别算法优化
- **BC形态**：识别价格突破+净量暴增+DELTA加速向上
- **SC形态**：识别价格暴跌+净空爆发+DELTA加速向下
- **LPS形态**：识别价格企稳S2上方3根K线+净多稳定+DELTA平坦/向上
- **LPSY形态**：识别价格企稳R2下方+净空主导/转空+DELTA向下/平坦

#### ⏳ 顶部/底部形态集成
- 创建TopPatternDetector类（BC/SC/LPS/LPSY）
- 创建BottomPatternDetector类（LPS/SC）
- 集成到WKF分析器，形成双重风控

## 三、技术亮点

### 3.1 图像识别技术
- **多维度融合**：OCR + 颜色分析 + 边缘检测 + 霍夫变换
- **自适应阈值**：根据实际像素分布动态调整颜色阈值
- **区域分割**：主图（0-45%）、净量（45-72%）、DELTA（72-100%）

### 3.2 Y坐标到价格映射
- **独立模型**：每个周期一个线性回归模型
- **高精度**：R²值0.25-0.69（15分钟和1小时最准确）
- **自适应**：根据人工标注数据不断校准

### 3.3 多周期共振算法
- **评分机制**：每周期20分，满分100分
- **优先级**：周线派发 > 日线派发 > 其他周期
- **风控策略**：≥80分有效，<80分无效

### 3.4 顶部/底部形态识别
- **形态特征**：结合价格结构+净量状态+DELTA斜率
- **双重风控**：周期阶段+形态识别，降低误判率
- **实战应用**：
  - 底部形态（SC/LPS）= 建仓/入场时机
  - 顶部形态（BC/LPSY）= 减仓/离场时机
