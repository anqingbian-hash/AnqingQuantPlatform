### 技术选型
1. **Spring Boot + Vue 3**：成熟稳定的技术栈，适合企业级应用
2. **MySQL 8.0**：支持 JSON 字段，存储灵活
3. **JPA/Hibernate**：简化数据库操作，提高开发效率
