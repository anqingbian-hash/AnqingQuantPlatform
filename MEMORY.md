- **系统版本**：V1.2

### 核心规则

#### WKF系统第一铁律
1. **周线派发 = 全局无效**
   - 全共振评分直接计0分
   - 风控建议统一为「0%仓位、观望」
   - 无条件执行，无例外

2. **日线派发 = 小周期无效**
   - 1小时/15分钟/5分钟所有信号一律无效
   - 全共振评分最高计20分（仅周线分值）
   - 风控建议为「0%仓位、观望」
   - 严格执行，无主观调整

#### 多周期判定
- **五周期**：周线、日线、1小时、15分钟、5分钟
- **各20分**：吸筹期/拉升期 = 20分/周期；派发期 = 0分/周期
- **信号有效性**：
  - ≥80分：高确定性全共振（有效）
  - 60-79分：弱共振（无效，按无效处理）
  - <60分：无共振（无效）

#### 各阶段判定规则

**吸筹期**（必须同时满足所有条件）：
- 价格结构：跌幅≥50%（股票）/ ≥30%（其他），站稳S2≥3根K线，箱体振幅≤8%
- NetVolume：净多环比增长≥10%，净空增幅≤5%
- DeltaBar：动能斜率向上，无价格与动能背离

**拉升期**（必须同时满足所有条件）：
- 价格结构：突破箱体≥3%，回踩不跌破箱体上沿，未跌破S2支撑位，高点持续抬升
- NetVolume：净多持续稳定，无由多转空现象
- DeltaBar：动能持续正值，动能斜率向上，无价格与动能顶背离

**派发期**（满足任意一条或默认）：
- 跌破S2支撑、高点拐头向下、震荡振幅扩大、净空主导、净多持续萎缩、净空爆发式增长、动能斜率向下、价格与动能顶背离、努力没结果（放量不涨）

## 专业术语对照

### WKF系统专业术语
- **LPS（最后的支撑点）**：二买位置，低吸参考
- **LPSY（最后的供应点）**：二卖位置，高抛参考
- **BC（购买高潮）**：短期高点，别追多
- **SC（卖出密集区）**：短期低点，别追空
- **SR线（动态滤网）**：红线=压力位，绿线=支撑位
- **净量**：实时"做多资金"和"做空资金"的差额
- **DELTA**：资金量的实时走势线
- **共振突破**：价格+净量+DELTA同时满足3个条件
- **四不开仓原则**：小白保命法则

### 威科夫三大定律
1. **因果原理**：通过横盘筹码计算未来行情大小
2. **供需原理**：供<求涨，供>求跌
3. **量价不协调原理**：预警趋势变化

### 数字净量系统四大组成部分
1. 主图-形态折线：判断BC、SC、TEST
2. 主图-动态滤网：SR线（支撑/压力位）
3. 副图1-数字净量：净多/净空状态及数值
4. 副图2-DELTA BAR：净量走势图，共振/背离/努力没结果
